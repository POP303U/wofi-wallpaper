<div align="center"> 
<h2> wofi-wallpaper </h2>
Wallpaper changer script for wofi
</div>

## Installation

```
git clone https://github.com/POP303U/wofi-wallpaper.git
cd wofi-wallpaper
sudo cp wofi-wallpaper /bin
```

### Hyprland keybinding

```
bind $mainMod, B, exec, wofi-wallpaper
```
